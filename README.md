# R&B Labs Enterprise

Landing institucional da R&B Labs, software house focada em sistemas sob medida, automacao e IA aplicada a operacoes reais.

## O que este projeto mostra

- Posicionamento comercial para empresas que precisam digitalizar processos.
- Estrutura de landing page com hero, prova social, dores, metodologia, comparacao, solucoes, FAQ e CTA.
- Modal de diagnostico para captacao de leads.
- Interface moderna com animacoes leves e componentes reutilizaveis.

## Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4
- Framer Motion
- Resend
- Lucide React

## Como rodar

```bash
npm install
npm run dev
```

Abra `http://localhost:3000` no navegador.

## Contexto

Este projeto faz parte da vitrine comercial da R&B Labs e apoia a mesma linha de produto do Wiver: tecnologia pratica, automacao e inteligencia aplicada a negocios locais e pequenas operacoes.
